让 AI 编程代理自主驾驶。

一行命令,代理就开始永续循环。每个阶段强制安全,中断后自动恢复。支持 Claude Code、Codex、Cursor、Gemini 等 40+ AI 编程代理。

$ npx skills add PresentJay/autopilot-skills

它做什么

你给一行任务描述,代理就跑 discover → analyze → plan → execute → verify → learn 循环,然后睡到下一轮。直到你叫停,或者任务完成。

它不做什么

  • 不直接 push 到 main 或 master。
  • 不触碰未授权的 .env、密钥、基础设施、workflow。
  • 允许路径外不执行 --force、--no-verify、git reset --hard、rm -rf。
  • 不重试已学习失败的模式 (NOT-OK 自动累积)。

节奏由你掌控

从 immediate · 2m · 5m · 15m (默认) · 30m · 1h+ 中选。1h+ 自动升级为 cron,会话关闭后仍能运行。空闲循环累积时,节奏自动放慢。

关键时刻被保留

日志写入 .autopilot.log/ (位置可改),PR 合并、bounded 任务完成、首次 zero-defect 这类有意义的事件,会自动晋升到 .autopilot.milestones/(固定路径,不可变)。

中断后会自我修复

主机 sleep、会话崩溃、wakeup 丢失等导致循环停下,下次调用时会自动诊断、清理工作分支,然后开始新循环。也可用 /autopilot status 单独查看当前状态。
