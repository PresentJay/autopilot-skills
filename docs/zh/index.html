面向 AI 编程代理的自主任务运行器。

一条命令。永续循环。每个阶段都强制安全。可在 Claude Code、Codex、Cursor、Gemini 以及其他 40+ AI 编程代理上运行。

$ npx skills add PresentJay/autopilot-skills

它做什么

你用一行文字交付任务,它运行 discover → analyze → plan → execute → verify → learn 自主循环,然后休眠到下一次循环 — 直到你叫停,或任务完成。

它不做什么

  • 不直接 push 到 main / master。
  • 不触碰你未授权的 .env、密钥、基础设施或 workflow。
  • 在允许路径外不运行 --force、--no-verify、git reset --hard、rm -rf。
  • 不重试已学习失败的模式 (NOT-OK 自动累积)。

节奏由你掌控

选择你想要的激进程度: immediate · 2m · 5m · 15m (默认) · 30m · 1h+ (自动升级为 cron)。空闲循环会自动减速。

里程碑被保留

日志位于 .autopilot.log/ (可变)。重要时刻 — PR 合并、bounded 任务完成、首次 zero-defect — 自动晋升到 .autopilot.milestones/,该路径固定且不可变。
