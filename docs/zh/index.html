让 AI 编程代理自己跑起来。

给一行任务,代理就一直跑下去 —— 直到你叫停,或者任务完成。每一步都把关安全,中断了也能自动接着跑。Claude Code、Codex、Cursor、Gemini 等 40 多种 AI 编程工具上都能用。

$ npx skills add PresentJay/autopilot-skills

它做什么

你给一行任务描述,代理就跑 discover → analyze → plan → execute → verify → learn 这一整圈,然后睡到下一轮再接着跑。直到你叫停,或者任务完成。

它不做什么

  • 不直接 push 到 main 或 master。
  • 没你授权的 .env、密钥、基础设施、workflow,一律不碰。
  • 允许路径以外,不跑 --force、--no-verify、git reset --hard、rm -rf。
  • 已经学到失败的模式,不再重试 (NOT-OK 自动累积)。

节奏你说了算

从 immediate、2m、5m、15m(默认)、30m、1h+ 里选。1h+ 自动升级成 cron,关掉会话也照样跑。空跑攒多了,节奏会自动放慢。

关键时刻另存一份

日志写在 .autopilot.log/(位置可以改);PR 合并、bounded 任务完成、首次 zero-defect 这种关键事件,自动晋升到 .autopilot.milestones/(这个路径是固定的、改不了)。

断了能自己接上

主机 sleep、会话崩溃、wakeup 丢了 —— 下次调用时它会自己诊断、清理工作分支,然后继续新一轮。只想看现在状态用 /autopilot status。
