# 节奏菜单

| 选项 | 实际延迟 | 说明 |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup 下限) | 最激进,令牌成本最高 |
| `2m` | 120s | 在 Anthropic 提示词缓存窗口内 — 高效 |
| `5m` | 300s | 略超出缓存窗口 |
| `15m` | 900s | **默认** — 平衡 |
| `30m` | 1800s | 宽松 |
| `1h+` | 3600s+ | 自动升级到 `/schedule` cron (cloud) |
| `manual` | — | 不自动唤醒,由你重新调用 |

连续 3 个空循环时,节奏会自动加慢一档。
