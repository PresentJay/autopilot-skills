# 节奏菜单

| 选项 | 实际延迟 | 什么时候? |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup 下限) | 给循环做压力测试,紧急清积压,或边看边跑的时候 |
| `2m` | 120s | 正在干活的会话 —— 反馈快,还在 Anthropic 缓存窗口里 |
| `5m` | 300s | 轻度迭代。比 2m 便宜,稍微出了缓存窗口 |
| `15m` | 900s | **默认**。偶尔回头看一眼的长跑任务 |
| `30m` | 1800s | 空跑常发生的任务。拉长间隔,空跑代价更低 |
| `1h+` | 3600s+ | 关掉会话也能跑。自动升到 `/schedule` cron (cloud) |
| `manual` | — | 演示、调试,或者想一轮一轮自己控制 |

连续 3 个空循环时,节奏会自己慢一档。
