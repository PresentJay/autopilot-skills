Self-driving mission runner for AI coding agents.

One command. Endless cycle. Safety enforced every phase. Works on Claude Code, Codex, Cursor, Gemini, and 40+ other harnesses.

$ npx skills add PresentJay/autopilot-skills

What it does

You hand it a mission in plain text. It runs an autonomous discover → analyze → plan → execute → verify → learn cycle and sleeps until the next one — until you tell it to stop, or until the mission is complete.

What it won't do

  • Push to main / master directly.
  • Touch .env, secrets, infra, or workflows you didn't allow.
  • Run --force, --no-verify, git reset --hard, or rm -rf outside allow paths.
  • Re-attempt patterns it has already learned to fail (NOT-OK auto-grow).

Cadence is yours

Pick how aggressive: immediate · 2m · 5m · 15m (default) · 30m · 1h+ (auto-promotes to a cron). Idle cycles auto-throttle.

Milestones, kept

The journal lives in .autopilot.log/ (mutable). Significant moments — PR merged, bounded mission complete, first zero-defect — auto-promote into .autopilot.milestones/, which is fixed and immutable.
