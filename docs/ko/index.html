AI 코딩 에이전트를 자율주행으로 운영하세요.

미션을 한 줄로 건네면, 멈추라 할 때까지 사이클이 돕니다. 모든 단계에서 안전 강제, 인터럽트 후 자가복구. Claude Code · Codex · Cursor · Gemini 등 40개 이상 AI 하니스에서 동작.

$ npx skills add PresentJay/autopilot-skills

무엇을 합니까

한 줄짜리 미션을 받으면 discover → analyze → plan → execute → verify → learn 자율 사이클을 돕니다. 사이클이 끝나면 다음 사이클까지 잠들고, 다시 깨어나 작업을 이어 갑니다. 멈추라 할 때까지, 또는 미션이 끝날 때까지.

하지 않는 것

  • main, master 직접 푸시 금지.
  • 허용하지 않은 .env, 비밀, 인프라, 워크플로우는 절대 건드리지 않음.
  • 허용 경로 밖에서 --force, --no-verify, git reset --hard, rm -rf 실행 금지.
  • 한 번 실패가 학습된 패턴은 다시 시도하지 않음 (NOT-OK 자동 누적).

주기는 직접 정하세요

immediate · 2m · 5m · 15m (기본값) · 30m · 1h+ 중 선택. 1h+ 는 cron 으로 자동 승격되어 세션이 닫혀도 동작합니다. 빈 사이클이 누적되면 주기는 자동으로 느려집니다.

중요 순간은 보존됩니다

저널은 .autopilot.log/ 에 쌓이고 (위치 변경 가능), PR 머지 · 미션 완료 · 첫 zero-defect 같은 의미 있는 사건은 .autopilot.milestones/ (고정·불변) 에 자동 승격됩니다.

인터럽트 후에도 알아서 복구

호스트 sleep, 세션 크래시, 누락된 wakeup 등으로 사이클이 중단되어도, 다음 호출 때 자동으로 진단하고 작업 브랜치를 정리한 뒤 새 사이클을 돕니다. /autopilot status 로 현재 상태만 점검할 수도 있습니다.
