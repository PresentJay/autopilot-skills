AI 코딩 에이전트, 알아서 돌게.

한 줄짜리 미션만 주세요. 멈추라 할 때까지 사이클이 알아서 돕니다. 단계마다 안전 가드, 끊겨도 자동으로 이어 갑니다. Claude Code · Codex · Cursor · Gemini 등 40여 종 AI 도구에서 동작.

$ npx skills add PresentJay/autopilot-skills

무엇을 하느냐

미션을 받으면 discover → analyze → plan → execute → verify → learn 사이클을 자율로 돌립니다. 한 사이클 끝나면 다음 사이클까지 잠들고, 다시 깨어 작업을 이어 갑니다. 멈추라 할 때까지, 또는 미션이 끝날 때까지.

건드리지 않는 것

  • main, master 에는 직접 push 안 함.
  • 허용 안 한 .env, 비밀, 인프라, 워크플로우는 절대 손대지 않음.
  • 허용 경로 밖에서 --force, --no-verify, git reset --hard, rm -rf 안 씀.
  • 한 번 실패한 패턴은 다시 시도 안 함 (NOT-OK 자동 누적).

주기는 직접 정해

immediate · 2m · 5m · 15m (기본) · 30m · 1h+ 중 골라 쓰세요. 1h+ 는 cron 으로 자동 승격되어 세션을 닫아도 계속 돕니다. 빈 사이클이 쌓이면 주기는 알아서 느려집니다.

중요한 순간은 따로 보관

저널은 .autopilot.log/ 에 쌓이고 (위치 변경 가능), PR 머지 · 미션 완료 · 첫 zero-defect 같은 핵심 사건은 .autopilot.milestones/ 에 자동 승격됩니다 (이쪽은 고정·불변).

끊겨도 알아서 복구

호스트 sleep, 세션 크래시, 누락된 wakeup 으로 사이클이 멈춰도 다음 호출 때 알아서 진단하고 작업 브랜치를 정리한 뒤 새 사이클을 시작합니다. 상태만 보고 싶을 땐 /autopilot status.
