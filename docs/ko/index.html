무엇을 하나

당신이 평문 한 줄로 미션을 건네면, discover → analyze → plan → execute → verify → learn 자율 사이클을 돌리고 다음 사이클까지 잠들었다가 깨어납니다 — 멈추라 할 때까지, 또는 미션이 완료될 때까지.

하지 않는 것

  • main / master 에 직접 push 금지.
  • 허용하지 않은 .env, secrets, infra, workflow 접근 금지.
  • 허용 경로 밖에서 --force, --no-verify, git reset --hard, rm -rf 실행 금지.
  • 이미 실패가 학습된 패턴 재시도 금지 (NOT-OK auto-grow).

주기는 당신의 것

공격적 정도를 직접 선택: immediate · 2m · 5m · 15m (기본값) · 30m · 1h+ (cron 자동 승격). 유휴 사이클이 누적되면 자동으로 주기가 느려집니다.

마일스톤은 보존

저널은 .autopilot.log/ (가변)에 쌓이고, 의미 있는 사건들 — PR 머지, bounded 미션 완료, 첫 zero-defect 달성 — 은 .autopilot.milestones/ (고정·불변)에 자동 승격됩니다.
