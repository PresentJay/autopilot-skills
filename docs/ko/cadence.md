# 주기 메뉴

| 선택지 | 실제 지연 | 설명 |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup 최소 단위) | 가장 공격적. 토큰 많이 씀 |
| `2m` | 120s | Anthropic 프롬프트 캐시 안 — 효율적 |
| `5m` | 300s | 캐시 살짝 벗어남 |
| `15m` | 900s | **기본** — 무난 |
| `30m` | 1800s | 여유 있게 |
| `1h+` | 3600s+ | `/schedule` cron 으로 자동 승격 (cloud) |
| `manual` | — | 자동 wake 안 함. 사용자가 직접 호출 |

빈 사이클이 연속 3번이면 주기는 한 단계 자동으로 느려집니다.
