# 주기 메뉴

| 선택지 | 실제 지연 | 언제? |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup 최소 단위) | 사이클 스트레스 테스트, 급한 백로그 한 번에 처리, 직접 보면서 굴릴 때 |
| `2m` | 120s | 활발한 작업 세션 — 피드백 빠르게, Anthropic 캐시 윈도우 안 |
| `5m` | 300s | 가벼운 반복. 2m 보다 저렴하고 캐시 윈도우 살짝 벗어남 |
| `15m` | 900s | **기본**. 가끔 들여다 보는 장기 미션 |
| `30m` | 1800s | 빈 사이클이 자주 나는 미션. 간격을 넓혀 빈 사이클 비용 줄이기 |
| `1h+` | 3600s+ | 세션 닫혀도 동작. `/schedule` cron 으로 자동 승격 (cloud) |
| `manual` | — | 데모, 디버깅, 사이클마다 직접 통제하고 싶을 때 |

빈 사이클이 연속 3번이면 주기는 한 단계 자동으로 느려집니다.
