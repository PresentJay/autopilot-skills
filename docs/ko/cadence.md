# 주기 메뉴

| 선택지 | 실제 지연 | 메모 |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup 최소 단위) | 가장 공격적, 토큰 소모 큼 |
| `2m` | 120s | Anthropic 프롬프트 캐시 윈도우 안 — 효율적 |
| `5m` | 300s | 캐시 윈도우 살짝 초과 |
| `15m` | 900s | **기본값** — 균형 |
| `30m` | 1800s | 여유 있게 |
| `1h+` | 3600s+ | `/schedule` cron 으로 자동 승격 (cloud) |
| `manual` | — | 자동 wake 없음. 사용자가 직접 재호출 |

빈 사이클이 연속 3번이면 주기는 자동으로 한 단계 느려집니다.
