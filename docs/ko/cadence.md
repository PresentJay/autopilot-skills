# 주기 메뉴

| 선택지 | 실제 지연 | 설명 |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup 최소 단위) | 가장 공격적; 토큰 비용 큼 |
| `2m` | 120s | Anthropic 프롬프트 캐시 윈도우 안 — 효율적 |
| `5m` | 300s | 캐시 윈도우 살짝 초과 |
| `15m` | 900s | **기본값** — 균형 |
| `30m` | 1800s | 여유 |
| `1h+` | 3600s+ | `/schedule` cron 자동 승격 (cloud) |
| `manual` | — | 자동 wake 없음; 사용자가 직접 재호출 |

유휴 자동 throttle: 연속 3 사이클이 비면 주기를 1단계 늘립니다.
