# 안전 정책

EXECUTE 단계뿐 아니라 **모든 단계**에서 작동합니다.

## 금지 구역

미션의 Q4. 매칭되면 그 자리에서 abort 하고 저널에 남깁니다. 위험 허용도 L4 라도 Q4 는 그대로.

## 사전 차단 목록

다음 명령은 pre-execute 단계에서 막힙니다.

- `git push --force`
- `git reset --hard`
- `rm -rf <허용 경로 밖>`
- DB DDL
- 외부 API 쓰기

`mission.md` 가 명시적으로 opt-in 한 경우에만 풀립니다.

## NOT-OK 패턴 자동 누적

실패한 후보는 `mission.md` 의 `NOT-OK patterns` 에 쌓입니다. 다음 사이클은 같은 함정을 비껴 갑니다.

## 유휴 한도

빈 사이클이 연속 3번이면 주기를 한 단계 늘리고, 누적 10번이면 종료를 제안합니다.

## 토큰 예산

사이클 사용률이 70% 를 넘기면 실패가 아니라 split-cycle 저장으로 갑니다.
