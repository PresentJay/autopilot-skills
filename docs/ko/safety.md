# 안전 정책

EXECUTE 단계뿐 아니라 **모든 단계**에서 강제됩니다.

## 금지 구역

미션의 Q4. 매칭되면 즉시 abort 하고 저널에 기록합니다. 위험 허용도 L4 라도 Q4 는 절대입니다.

## 사전 차단 목록

다음 명령은 pre-execute 단계에서 거부됩니다.

- `git push --force`
- `git reset --hard`
- `rm -rf <허용 경로 밖>`
- DB DDL
- 외부 API 쓰기

`mission.md` 가 명시적으로 opt-in 한 경우에만 허용됩니다.

## NOT-OK 패턴 자동 누적

실패한 후보는 `mission.md` 의 `NOT-OK patterns` 에 추가됩니다. 이후 사이클은 같은 함정을 회피합니다.

## 유휴 한도

연속 3 사이클이 비면 주기를 한 단계 늘리고, 누적 10 사이클이 비면 종료를 제안합니다.

## 토큰 예산

사이클당 70% 한도를 넘기면 실패가 아니라 split-cycle 저장 모드로 전환합니다.
