はじめに ミッション セーフティ サイクル マイルストーン GitHub EN 한 中 日

AI コーディングエージェントを、自律走行に。

一行のミッションを渡せば、止めるまでサイクルが回り続けます。すべてのフェーズで安全を強制、中断後も自動で復旧。Claude Code・Codex・Cursor・Gemini ほか 40+ AI ハーネスで動作。

$ npx skills add PresentJay/autopilot-skills

何をするか

一行のミッションを渡すと、discover → analyze → plan → execute → verify → learn の自律サイクルを回し、次のサイクルまでスリープします。停止指示があるか、ミッションが完了するまで。

やらないこと

  • main・master への直接 push をしません。
  • 許可していない .env、シークレット、インフラ、ワークフローには触れません。
  • 許可パス外で --force・--no-verify・git reset --hard・rm -rf を実行しません。
  • 失敗が学習されたパターンは再試行しません(NOT-OK 自動蓄積)。

サイクルはあなた次第

immediate・2m・5m・15m(デフォルト)・30m・1h+ から選択。1h+ は cron に自動昇格し、セッションを閉じても動き続けます。アイドルが続けばサイクルは自動で減速します。

重要な瞬間は保存

ジャーナルは .autopilot.log/ に蓄積され(場所は変更可)、PR マージ・bounded ミッション完了・初回 zero-defect といった意味のある瞬間は .autopilot.milestones/(固定・不変)に自動昇格されます。

中断後も自動で復旧

ホスト sleep・セッションクラッシュ・wakeup の取りこぼしでサイクルが止まっても、次の呼び出し時に自動で診断し、作業ブランチを片付けてから新しいサイクルを始めます。/autopilot status で現在の状態だけ確認することもできます。
