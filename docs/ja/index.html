AIコーディングエージェントのための自律走行ミッションランナー。

1つのコマンド。終わりなきサイクル。すべてのフェーズでセーフティを強制。Claude Code、Codex、Cursor、Gemini ほか 40+ AI ハーネスで動作。

$ npx skills add PresentJay/autopilot-skills

何をするか

あなたが平文で 1 行のミッションを渡すと、discover → analyze → plan → execute → verify → learn の自律サイクルを回し、次のサイクルまでスリープします — 停止指示があるか、ミッションが完了するまで。

やらないこと

  • main / master への直接 push をしない。
  • 許可していない .env、シークレット、インフラ、ワークフローには触れない。
  • 許可パス外で --force、--no-verify、git reset --hard、rm -rf を実行しない。
  • 失敗が学習済みのパターンを再試行しない (NOT-OK 自動蓄積)。

サイクルはあなた次第

積極度を選択: immediate · 2m · 5m · 15m (デフォルト) · 30m · 1h+ (cron に自動昇格)。アイドルが続くと自動で減速します。

マイルストーンは保存

ジャーナルは .autopilot.log/ (可変)に蓄積され、重要な瞬間 — PR マージ、bounded ミッション完了、初めての zero-defect — は .autopilot.milestones/ (固定・不変)に自動昇格されます。
