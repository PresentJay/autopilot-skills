AI エージェントを、走らせ続ける。

一行のミッションを渡すだけ。あなたが止めるまで、サイクルが回り続けます。各フェーズで安全をガード、止まっても自動で再起動。Claude Code・Codex・Cursor・Gemini ほか 40 種類以上のコーディング AI で動きます。

$ npx skills add PresentJay/autopilot-skills

何をするか

一行のミッションを渡すと、discover → analyze → plan → execute → verify → learn という自律サイクルを回します。1 周終わったら次のサイクルまでスリープして、起きてからまた続きをやる。あなたが止めるまで、もしくはミッションが終わるまで。

やらないこと

  • main・master に直接 push しません。
  • 許可していない .env・シークレット・インフラ・ワークフローには触りません。
  • 許可パス外で --force・--no-verify・git reset --hard・rm -rf を実行しません。
  • すでに失敗を学習したパターンは、もう試しません(NOT-OK 自動蓄積)。

サイクルはあなた次第

immediate・2m・5m・15m(デフォルト)・30m・1h+ から選びます。1h+ は cron に自動昇格するので、セッションを閉じても動き続けます。空サイクルが続けば、テンポは自動で落ちます。

大事な瞬間は別建て

ジャーナルは .autopilot.log/ に貯まります(場所は変えられます)。PR マージ・bounded ミッション完了・初回 zero-defect のような重要な瞬間は、自動で .autopilot.milestones/ へ昇格します(こちらは固定で動かせません)。

止まっても自分で立ち上がる

ホスト sleep・セッションクラッシュ・wakeup の取りこぼしでサイクルが止まっても、次の呼び出しで自動で診断し、作業ブランチを片付けてから新しいサイクルを始めます。状態だけ確認したいときは /autopilot status。
