# セーフティポリシー

EXECUTE フェーズだけでなく、**どのフェーズでも**動いています。

## 禁止ゾーン

ミッションの Q4。引っかかった瞬間に abort して、ジャーナルに残します。リスク階層 L4 でも Q4 は譲りません。

## 事前ブロックリスト

下のコマンドは pre-execute 段階で弾かれます。

- `git push --force`
- `git reset --hard`
- `rm -rf <許可パス外>`
- データベースの DDL
- 外部 API への書き込み

`mission.md` で明示的に opt-in したときだけ通します。

## NOT-OK パターン自動蓄積

失敗した提案は `mission.md` の `NOT-OK patterns` に追加されます。次のサイクルからは同じ罠を避けます。

## アイドル上限

連続 3 サイクル空ならサイクルを 1 段階上げ、累計 10 サイクル空なら終了を提案します。

## トークン予算

サイクル中の使用率が 70% を超えたら、失敗扱いではなく split-cycle 保存に切り替えます。
