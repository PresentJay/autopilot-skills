# セーフティポリシー

EXECUTE フェーズだけでなく、**すべてのフェーズ**で強制されます。

## 禁止ゾーン

ミッションの Q4。マッチした瞬間に abort し、ジャーナルに記録します。リスク階層 L4 でも Q4 は絶対です。

## 事前ブロックリスト

次のコマンドは pre-execute 段階で拒否されます。

- `git push --force`
- `git reset --hard`
- `rm -rf <許可パス外>`
- DB の DDL
- 外部 API への書き込み

`mission.md` で明示的に opt-in した場合のみ許可されます。

## NOT-OK パターン自動蓄積

失敗した提案は `mission.md` の `NOT-OK patterns` に追加されます。後続サイクルは同じ罠を避けます。

## アイドル上限

連続 3 サイクルが空ならサイクルを 1 段階増やし、累計 10 サイクルが空なら終了を提案します。

## トークン予算

サイクル中の使用率が 70% を超えると、失敗扱いではなく split-cycle 保存に切り替わります。
