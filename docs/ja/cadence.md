# サイクルメニュー

| 選択肢 | 実効待機 | メモ |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup の下限) | 最も積極的。トークンコスト大 |
| `2m` | 120s | Anthropic プロンプトキャッシュ窓内 — 効率的 |
| `5m` | 300s | キャッシュ窓を少し超過 |
| `15m` | 900s | **デフォルト** — バランス |
| `30m` | 1800s | 余裕あり |
| `1h+` | 3600s+ | `/schedule` cron へ自動昇格 (cloud) |
| `manual` | — | 自動 wake なし。ユーザーが再度呼び出し |

連続 3 サイクルが空のときは、サイクルを 1 段階引き上げます。
