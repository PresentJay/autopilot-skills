# サイクルメニュー

| 選択肢 | 実効待機 | いつ? |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup の下限) | サイクルの負荷試験、急ぎのバックログ消化、見ながら回したいとき |
| `2m` | 120s | 集中作業中のセッション — 反応が速く、Anthropic キャッシュ窓内 |
| `5m` | 300s | 軽い反復。2m より安く、キャッシュ窓を少し外れた程度 |
| `15m` | 900s | **デフォルト**。ときどき様子を見る長期ミッション |
| `30m` | 1800s | 空サイクルが多いミッション。間隔を広げて空サイクルのコストを下げる |
| `1h+` | 3600s+ | セッションを閉じても動く。`/schedule` cron へ自動昇格 (cloud) |
| `manual` | — | デモ、デバッグ、自分で 1 サイクルずつ進めたいとき |

連続 3 サイクル空のときは、サイクルを自動で 1 段階下げます。
