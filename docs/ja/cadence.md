# サイクルメニュー

| 選択肢 | 実効待機 | メモ |
|---|---|---|
| `immediate` | 60s (ScheduleWakeup の下限) | いちばん攻める。トークンも食う |
| `2m` | 120s | Anthropic プロンプトキャッシュ窓内 — 効率的 |
| `5m` | 300s | キャッシュ窓を少し外れる |
| `15m` | 900s | **デフォルト** — バランス |
| `30m` | 1800s | 余裕あり |
| `1h+` | 3600s+ | `/schedule` cron へ自動昇格 (cloud) |
| `manual` | — | 自動 wake なし。自分で呼び出す |

連続 3 サイクル空のときは、サイクルを自動で 1 段階下げます。
